# Engineering

The engineering domain owns implementation-facing technical specifications.

## Current Areas

- [Data Model And Schema](data-model/index.md)

## Ownership

Primary owner: Engineering  
Required reviewers when changed: Product for product meaning, Governance for durable decisions, AI Quality when model behavior is affected.

## Belongs Here

- Data model and schema specs.
- Migration/change specs.
- Implementation-facing technical plans.
- Interfaces that platform code must satisfy.

## Does Not Belong Here

- Product roadmap prose.
- Prompt wording as source artifact.
- Publishing instructions.
