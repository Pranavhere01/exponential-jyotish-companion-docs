# Evaluation Artifacts

Status: Draft v0.1  
Date: 2026-06-29

This folder stores reusable evaluation rubrics, fixtures, and sample cases for model-router validation.

Evaluation goals:

- Validate open-source/open-weight Chinese model candidates behind the router before production use.
- Compare hosted and self-hosted models on the same tasks.
- Check groundedness, tool use, prediction delivery, safety, latency, and cost.
- Keep difficult or negative Jyotish content available when responsibly delivered.
